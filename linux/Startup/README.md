Step 1: Input your token in start.sh file:
```bash
TOKEN=YourAuthToken
```

Step 2: Running ERa with systemctl
```bash
$ cd ~/era-lib/linux/Startup
$ ./build.sh raspberry
```

Some tips that can be handy later:
```bash
# Stopping ERa
sudo systemctl stop era

# Starting ERa
sudo systemctl start era
```
